Steps to run this website on localhost:

(1) Install Python 3.12.3
(2) $pip install -r requirements.txt
(3) $git clone git@github.com:mmmckie/jhu_software_concepts.git
(4) Ensure current working directory is jhu_software_concepts/module_1
(5) $python run.py
(6) Navigate to 'http://localhost:8080/' or 'http://127.0.0.1:8080/', in your web
